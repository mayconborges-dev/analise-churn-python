# Análise de Cancelamento de Clientes (Churn)

Projeto de análise de dados desenvolvido em Python para identificar fatores que influenciam o cancelamento de clientes.

## Tecnologias utilizadas
- Python
- Pandas
- Plotly

## Objetivo
Identificar padrões de comportamento dos clientes e propor ações para reduzir a taxa de cancelamento.

## Resultado
Análise indicou possíveis estratégias que poderiam reduzir o churn de 56% para aproximadamente 18%.
